CS Url Shortner

CloudSpokes Url Shortener

When this service receives a URL that ends with an integer (e.g., http://www.csurlshortner.herokuapp.com/2021) it assume that it is for a challenge and redirect the user to http://www.cloudspokes.com/challenges/2021. When the service receives a URL that ends with a string (e.g., www.csurlshortner.herokuapp.com/jady) offcourse except "csurlshortnerabout" it assumes that it is for a member and redirect the user to http://www.cloudspokes.com/members/jady.
